# 金蚀物语 (Rustlight)

《金蚀物语》是一款休闲奇幻风格的文字冒险游戏，讲述穿越者主角在锈斑侵蚀的世界中，与悬灯村反抗军共同对抗教会暴政的故事。玩家将探索炼金灾难后的世界，通过独特的能力帮助亚人抑制锈蚀，在各个地点赚取金币，培养战力，提高声望，并随着与蛋挞亚人小嘉、鹿亚人鹿宝等角色的羁绊加深，逐步揭开"黄金锈斑"背后的真相。
